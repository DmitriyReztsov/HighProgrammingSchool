# OOAP_JAVA

Java версия проекта Object-Oriented Architecture and Programming (OOAP)

## Структура проекта

- `src/main/java/ooap/` — исходный код
- `test/java/ooap/` — юнит-тесты
- `docs/` — документация
- `scripts/` — скрипты компиляции

## Требования

- Java 11 или выше
- JUnit 5 (для тестов)

## Компиляция

```bash
./scripts/compile.sh
```

## Запуск тестов

```bash
./scripts/test.sh
```

## Очистка

```bash
./scripts/clean.sh
```

## Запуск класса

```bash
./scripts/run.sh ClassName
```
