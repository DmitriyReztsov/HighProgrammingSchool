классы реализации
CellNode - клетка как узел в графе игрового поля. Содержит значение и связи с соседями
PlayGroundGraph - игровое поле как набор узлов-клеток, имеющих связи друг с другом.
    Содержит ссылку на первый узел.
StepStack - обертка над стандартным списком из вводимых строк для учета шагов конкретного пользователя.
    В ТЗ нет разъяснения что именно необходимо хранить в виде "хода игрока". Возможно, отдельную
    структуру для "хода" в виде, например, состояния поля и ввода игрока можно будет создать позже.
Step - возможно, на будущее - шаг игрока. Состояние поля и ввод игрока по его изменению.
