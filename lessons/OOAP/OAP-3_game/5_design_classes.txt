StateABC - абстрактный класс, интерфейс для классов состояний игрового поля
PlayGroundState - класс или классы состояний для игрового поля
CellFabric - класс для создания и обновления ячеек
BonusBehaviorABC - абстрактный класс, интерфейс для поведения разных бонусов
BonusBehaviorCross - бонусы если элементы выстроились в крест (как расчитываются бонусы)
BonusBehaviorLine - для линии
... возможны другие бонусы, которые как-то по-своему будут менять состояние поля
